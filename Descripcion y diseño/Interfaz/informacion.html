Sistema operativo objetivo:
Descripción
Herramientas especializadas en atacar un sistema operativo concreto.

Tipos
Android: Android es un sistema operativo inicialmente pensado para teléfonos móviles, al igual que iOS, Symbian y Blackberry OS. Lo que lo hace diferente es que está basado en Linux, un núcleo de sistema operativo libre, gratuito y multiplataforma.

iOS: iOS es un sistema operativo móvil de la multinacional Apple Inc. Originalmente desarrollado para el iPhone (iPhone OS), después se ha usado en dispositivos como el iPod touch y el iPad. No permite la instalación de iOS en hardware de terceros.

OS-X: OS X es el sistema operativo detrás de cada Mac. Está diseñado sobre una sólida base UNIX para aprovechar el hardware al máximo. Es elegante y fácil de usar. Viene con una increíble colección de apps que te encantará usar todos los días.

Ubuntu: Ubuntu es un sistema operativo basado en GNU/Linux y que se distribuye como software libre, el cual incluye su propio entorno de escritorio denominado Unity. Su nombre proviene de la ética homónima, en la que se habla de la existencia de uno mismo como cooperación de los demás.

Windows: Microsoft Windows (conocido generalmente como Windows o MS Windows), es el nombre de una familia de distribuciones de software para PC, smartphone, servidores y sistemas empotrados, desarrollados y vendidos por Microsoft, y disponibles para múltiples arquitecturas, tales como x86 y ARM. Desde un punto de vista técnico, no son sistemas operativos, sino que contienen uno (tradicionalmente MS-DOS, o el más actual cuyo núcleo es Windows NT) junto con una amplia variedad de software; no obstante, es usual (aunque no necesariamente correcto) denominar al conjunto como sistema operativo en lugar de distribución.

Recopilación de información:
Descripción
Herramientas basadas en análisis que recogen información de todo tipo.

Tipos
Análisis de DNS: Nos puede servir para enumerar dominios con sus entradas DNS, o buscar información en la caché DNS de un servidor, desvelando en por qué páginas ha navegado.

Análisis de tráfico: Permite que veamos el tráfico de nuestra red local, para lo cual nuestra tarjeta de red deberá configurarse en modo promiscuo.

Análisis de OSINT: Inteligencia de fuentes abiertas u «Open Source Intelligence» (OSINT) hace referencia al conocimiento recopilado a partir de fuentes de acceso público. El proceso incluye la búsqueda, selección y adquisición de la información, así como un posterior procesado y análisis de la misma con el fin de obtener conocimiento útil y aplicable en distintos ámbitos. Constituyen una buena base para posteriores ataques de ingeniería social o phising.

Análisis de routeo: Permite que obtengamos información sobre máscaras de red ICMP, o sobre la trayectoria que siguen los paquetes que circulan por nuestra red.

Análisis de SMB: Server Message Block (SMB) es un protocolo de red que permite compartir archivos,impresoras, etc, entre nodos de una red de computadoras que usan el sistema operativo Microsoft Windows. Este protocolo pertenece a la capa de aplicación en el modelo OSI.

Análisis de SMTP: El Simple Mail Transfer Protocol (SMTP) o “protocolo para transferencia simple de correo”, es un protocolo de red utilizado para el intercambio de mensajes de correo electrónico entre computadoras u otros dispositivos (PDA, teléfonos móviles, impresoras, etc). Fue definido en el RFC 2821 y es un estándar oficial de Internet.

Análisis de SNMP: El Protocolo simple de administración de redes (SNMP) es un protocolo que se usa para administrar redes TCP/IP complejas. Con SNMP, los administradores pueden administrar y configurar equipos en red desde un equipo centralmente ubicado, en lugar de tener que ejecutar software de administración de red.

Análisis de SSL: SSL (Secure Socket Layer) es un proceso que administra la seguridad de las transacciones que se realizan a través de Internet. Se basa en un proceso de cifrado de clave pública que garantiza la seguridad de los datos que se envían a través de Internet. Su principio consiste en el establecimiento de un canal de comunicación seguro (cifrado) entre dos equipos (el cliente y el servidor) después de una fase de autenticación.

Análisis de VoIP: VoIP (Voice over IP), es un conjunto de recursos que hacen posible que la señal de voz viaje a través de Internet empleando el protocolo IP.

Análisis de VPN: Un VPN (Virtual Private Network) te permite crear una conexión segura a otra red a través del Internet. Cuando conectas cualquier dispositivo a un VPN, este actúa como si estuviese en la misma red que la que tiene el VPN y todo el tráfico de datos se envía de forma segura a través del VPN.

Detección de servicio: Nos permite conocer los servicios de los que dispone nuestra víctima, así como los puertos en los que se localizan.

Detección de SO: Nos permite conocer el nombre y la versión del sistema operativo que nuestra víctima está utilizando.
Identificación de hosts en línea: Nos sirve para recopilar información sobre los distintos dispositivos o hosts conectados a nuestra red.

Identificación de IDS: Permite descubrir la presencia de sistemas de detección de intrusos en nuestra red.

Análisis de vulnerabilidades:
Descripción
Herramientas basadas en escanear programas y bases de datos, o analizar configuraciones por defecto que puedan ser vulnerables.

Tipos
Herramientas Cisco: Nos permite recopilar información sobre routers Cisco, para poder evaluar sus vulnerabilidades, además de disponer de herramientas para explotar estos dispositivos.

Evaluación de BBDD: Nos permite recopilar información sobre bases de datos para evaluar vulnerabilidades y realizar ataques de SQLinjection.

Fuzzing: Se llama fuzzing a las diferentes técnicas de testeo de software capaces de generar y enviar datos secuenciales o aleatorios a una o varias áreas o puntos de una aplicación, con el objeto de detectar defectos o vulnerabilidades existentes en el software auditado. Es utilizado como complemento a las prácticas habituales de chequeo de software, ya que proporcionan cobertura a fallos de datos y regiones de código no testados, gracias a la combinación del poder de la aleatoriedad y ataques heurísticos, entre otros.

Evaluación de software: Realización de determinadas pruebas para comprobar el nivel de seguridad de un determinado software.

Otros escáneres: Recogen otro tipo de escáneres para complementar al resto ya comentados.

Herramientas de contraseñas:
Descripción
Herramientas que permiten realizar ataques de diccionario, fuerza bruta o rainbow tables para romper contraseñas que no sean muy elaboradas en poco tiempo.

Tipos
Herramientas para GPU: Unidad de procesamiento gráfico o GPU (Graphics Processor Unit) es un coprocesador dedicado al procesamiento de gráficos u operaciones de coma flotante, para aligerar la carga de trabajo del procesador central en aplicaciones como los videojuegos o aplicaciones 3D interactivas.

Ataques sin conexión: Permite que realicemos ataques de contraseñas aún sin disponer de conexión a Internet, mediante ataques de diccionario o de fuerza bruta.

Ataques con conexión: Realiza ataques de contraseña apoyándose en búsquedas a través de Internet sobre información acerca de la víctima para aumentar la eficiencia del ataque.

Herramientas wireless:
Descripción
Herramientas orientadas a sistemas Bluetooth, NFC o RFID, entre otros.

Tipos
Herramientas Bluetooth: Nos permite localizar y atacar a dispositivos Bluetooth

Herramientas RFID/NFC: RFID, o Identificación por Radiofrecuencia se utiliza para identificar un elemento, seguir su ruta de movimiento y calcular distancias gracias a una etiqueta especial que emite ondas de radio, la cual se adjunta o se encuentra incorporada al objeto. La tecnología RFID permite la lectura de etiquetas incluso cuando éstas no se encuentran en una línea visual directa y puede además penetrar finas capas de materiales (pintura, nieve, etc.). Por su parte, NFC, (comunicación de campo cercano en español) es una tecnología de comunicación inalámbrica, de corto alcance y alta frecuencia que permite el intercambio de datos entre dispositivos.

Herramientas WiFi: Sirven para identificar y lanzar ataques contra sistemas wireless como pueden ser WiFi o Zigbee.

Herramientas para exploit:
Descripción
Herramientas relacionadas con Metasploit y técnicas de ingeniería social.

Tipos
Denegación de servicio: También conocidos como DoS, permiten la sobrecarga de recursos computacionales del objetivo, provocando la caída del servicio de forma indefinida.

Explotación de redes: Herramientas de penetración utilizadas para aprovechar vulnerabilidades o fallos existentes en distintos equipos, de modo que puedan obtener acceso a ellos.

Ingeniería social: Herramientas que nos permiten recopilar información sobre nuestras víctimas para realizar un posterior ataque especialmente dirigido.

Herramientas de Sniffing/Spoofing/Forensics:
Descripción
Herramientas que permiten capturar paquetes, envenenamiento de redes o recuperación de archivos.

Tipos
Network sniffer: Permite capturar y analizar paquetes que circulan por la red, obteniendo todo tipo de información contenida en ellos e incluso permitiendo redirigir los paquetes a otro destino.

Network spoofing: Permiten suplantar la identidad de usuarios o servidores.

Herramientas VoIP: Herramientas especializadas en capturar e insertar modificaciones en comunicaciones realizadas a través de voz sobre IP.

Web sniffer: Funciona de la misma manera que el sniffer de red pero a un nivel lógico inferior, pues está especializado en capturar paquetes que se envían desde un navegador web al servidor.

Herramientas forense: Permite recuperar archivos perdidos o eliminados de nuestro ordenador.

Herramientas post-exploit:
Descripción
Conjunto de herramientas útiles tras haber realizado un ataque de explotación en una máquina, por lo que son conocidas como herramientas post-exploit.

Tipos
SO backdoor: Un backdoor es un tipo de troyano que permite el acceso al sistema infectado y su control remoto, a pesar de que la víctima cambie sus credenciales tras haber sido atacada. El atacante puede entonces eliminar o modificar archivos, ejecutar programas, enviar correos masivamente o instalar herramientas maliciosas.

Web backdoor: Misma funcionalidad que el backdoor de SO, pero orientado a servidores web.

Herramientas para túneles: Herramientas que facilitan la creación de túneles para conectar a dos equipos cualesquiera.

Herramientas de ingeniería inversa:
Descripción
Herramientas que permiten desensamblar y reconstruir aplicaciones.

Tipos
Depurador: Herramienta que permite al usuario ejecutar un programa paso a paso, y examinar varios valores y acciones.

Desensamblador: Esencialmente, un desensamblador es exáctamente lo contrario de un ensamblador. Tal como un ensamblador convierte código escrito en ensamblador en código máquina binario, un desensamblador invierte el proceso e intenta recrear el código en ensamblador partiendo del código máquina binario.

Otras herramientas: Herramientas complementarias con las ya descritas.

Presentación de informes:
Descripción
Herramientas especializadas gestión de evidencias y captura de medios.

Tipos
Gestión de evidencia: Herramienta que permite la generación de informes para facilitar el análisis de resultados.

Capturador de medios: Herramientas que permiten labores de utilidad como captura de pantalla, descarga de vídeos alojados en servidores, mostrar información sobre la conectividad de tu equipo...etc.

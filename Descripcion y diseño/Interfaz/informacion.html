Información


Bienvenido a la herramienta online de consulta de ataques de la distribución Kali Linux.

A través de esta aplicación usted podrá:

1. Obtener información sobre un ataque en concreto.

2. Medir el nivel de seguridad de una aplicación o sitio web concreto.

3. Obtener herramientas necesarias para realizar un ataque, teniendo claro cual será el objetivo.


Instrucciones:

Dispone del siguiente formulario para realizar búsquedas de ataques según distintos parámetros:

- Lenguaje en el que están desarrollados.

- Protocolo objetivo.

- Ataques cuyo objetivo sea el de reconocimiento de redes.

- Ataques cuyo objetivo sea el de analizar vulnerabilidades de aplicaciones web.

- Ataques cuyo objetivo sea el de obtener información como contraseñas o archivos confidenciales.
